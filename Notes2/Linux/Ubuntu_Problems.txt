====== Ubuntu Problems ======
Created Monday 04 August 2025

=== System disk getting automatically filled up after broken install ===

**Checklist:**
sudo apt clean
sudo rm -rf /var/cache/apt/archives/*

Check huge folders:
sudo du -xh / | grep -P 'G\t' | sort -rh | head -20

Check for logs:
sudo journalctl --disk-usage

Check for held deleted files:
sudo lsof | grep deleted
