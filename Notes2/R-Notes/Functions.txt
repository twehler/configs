=== vapply: ===
vapply() is similar to sapply() but allows you to explicitly specify the output type, making it safer and faster.
vapply(X, FUN, FUN.VALUE, USE.NAMES = FALSE), where:

X: Input list/vector (generation_list here).
FUN: Function to apply to each element.
FUN.VALUE: A template for the output type (character(1) means each output is a single string).
USE.NAMES = FALSE: Skips assigning names to the output (for cleaner results).
