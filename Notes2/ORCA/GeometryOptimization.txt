====== GeometryOptimization ======
Created Thursday 13 November 2025

1. Craft Molecule in Avogadro, optimize geometry there (acts as a starting point)
2. Export as .xyz
3. Rename file to .inp
4. Take the raw coordinates and write them like the following, where the first line describes the used model and functional set:


!PBE D4 DEF2-SVP OPT		# functions
* xyz 0 1					# first number is charge, second is singlet
N         -0.83911        0.76325       -0.31843
C          0.61442        0.72014       -0.25075
C          1.01669       -0.56167        0.49740
O          0.20095       -1.36984        0.93753
H         -1.37884        0.05803        0.17605
H          1.00414        0.66192       -1.27362
C          1.17285        1.95192        0.45211
H          0.87124        2.87150       -0.05988
H          0.81191        2.01288        1.48492
H          2.26726        1.92903        0.48485
H         -1.30551        1.57618       -0.71069
O          2.33980       -0.77979        0.66176
H          3.04559       -0.08055        0.28096
*

5. $ orca file.inp > file.out
