====== Ollama ======
Created Friday 11 July 2025

=== Install ===

**Install Ollama on system partition first and pull a very small model to create all necessary folders:**
curl -fsSL https://ollama.com/install.sh | sh
ollama serve
ollama pull tinyllama:1.1b

**Show all ollama folders and how much space they are using:**
.1b
sudo du -ahx / 2>/dev/null | grep ollama | grep -E 'G|M' | sort -h

→ now move these folders to bigger partition with symlinks (requires full rights on the bigger partition)

→ run models with:
ollama run <model name>

sudo systemctl daemon-reload
sudo systemctl restart ollama


=== Models: ===
tinyllama:1.1b
deepseek-r1:7b
mistral:7b

**Move Ollama directory to new partition:**
sudo mv /usr/share/ollama/.ollama/ [[/mnt/sda3/data1/ollama/]]
sudo ln -s /mnt/sda3/data1/ollama/.ollama/ /usr/share/ollama/.ollamasystemctl --user restart ollama
